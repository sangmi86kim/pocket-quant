## 제미나이 연구원 원문 (요지)

**1. 통계·방법론** [확실 톤]
- 시드 **5,000~10,000개 이상 필수** — 꼬리(하위 1~5%) 분포 수렴엔 수천 번 몬테카를로 필요.
- 다중검정 + 생존편향 "100% 발생" — 파레토 후보 전부 같은 시드셋에 대입하면 '럭키 뽀록' 1등이
  반드시 생김. 챔피언끼리만 올리면 robust가 '가짜'.

**2. 합성 QLD 수학 오류** [확실 톤]
- 레버리지 ETF는 **일별 단순수익률의 2배**(=로그로 `ln(1+2R)`)를 추종. "로그수익률 ×2"는
  `2·ln(1+R) = ln((1+R)²) = ln(1+2R+R²)` → **R²만큼 매일 누적 오차** → 실제보다 유리하게(decay 약화).
- 처방: **단순수익률로 2배 계산 후 로그 변환**. 차입비용(스왑 마진) 빠짐 — 닷컴기 고금리에서 특히 큼.

**3. 상대평가 함정** [확실 톤]
- 양극단 기준선(현금/풀매수)이 시드마다 순위 널뛰기 → 전략이 방어 잘해 상위인지, 극단 봇이 자멸해
  어부지리인지 구분 불가. **순위(Ordinal) 말고 기준선과의 거리(Metric)로 측정.**

**4. 지표** [추정 톤]
- `score_vs_dca` 스칼라 합산 = 다목적(NSGA-III) 취지 훼손 = 단일목적 회귀.
- 대안: **상방/하방 캡처 비율(Capture Ratio)** · **CVaR**.

**5. 적대적 시나리오** [확실 톤]
- ① **동적 decay 지옥**: 지수는 제자리인데 매일 +1%/−1% 무한 반복으로 레버리지 녹이는 횡보 시드 강제 주입.
- ② **상관관계 붕괴**: 모든 자산 동시 박살 국면 인위 설계.

**6. 맹점** [확실 톤]
- 21일 블록은 수개월~수년 구조적 트렌드 유실(2001 폭락 토막 뒤 2020 급등 토막 = '가짜 변동성 갭').
- 처방: **블록을 최소 63거래일(3분기)로 늘리거나 regime-switching**.

## Opus 연구원 검증 + 종합

### ⚠️ §2 수학 직접 검증 → **제미나이 정확** (거짓말 아님)
- 레버리지 ETF 일별 로그수익률 = `ln(1+2R)`. 우리 프롬프트 'A방식'(로그×2) = `2·ln(1+R) = ln(1+2R+R²)`.
- 차 = `ln(1+2R+R²) − ln(1+2R) ≈ R² > 0` → **A는 실제보다 매일 R²만큼 높음 = decay 약화 = 실물보다
  유리한 가짜 지옥**. 제미나이 결론 맞음(클로드와 일치, **3중 교차**).
- 제미나이 처방 `ln(1+2R)` 누적 = 클로드 B방식(`×(1+2R) 일별복리`)과 **수학적으로 동일**. ✅

### 3중 교차검증 (GPT+클로드+제미나이 합의 = 확정)
1. 합성 QLD = **B방식**(단순수익률×2 일별복리), A(로그×2)는 R² 오차로 decay 약화 → 금지.
2. **블록 부트스트랩이 거시 트렌드/장기 자기상관 파괴** (+우리 lab 발견 = 4중).
3. **순위 말고 거리/꼬리지표**(CVaR), score_vs_dca 스칼라 자기모순.
4. 다중검정/생존편향 → 홀드아웃 분리.
5. 적대적 시나리오(상관붕괴 등).

### 제미나이 신규 기여 (반영)
- **동적 decay 지옥 시드(+1%/−1% 횡보)** — decay를 *직접* 겨냥하는 구체 적대시드. (클로드의 "decay는
  지옥 안 만들어도 직접 시험 가능"을 구체화) ⭐
