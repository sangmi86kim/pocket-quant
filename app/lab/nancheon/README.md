# 난천 (Nancheon) 연구 — QLD 2배 지옥 최종보스 시험장

> 챔피언로드 구조: 사천왕(hold-out) → **난천(챔피언전)** → 배틀 프론티어(엔드게임).
> 난천 = 시험마다 랜덤 시드로 생성하는 "QLD(나스닥100 2배 레버리지) 평행세계"에서
> 챔피언이 성실이(DCA)·기준선들 대비 **어떤 국면이 와도 덜 무너지나(robust)** 보는 무대.
> ⚠️ **설계 미확정** — 다중 AI 연구원 교차검토 중 (reviews/).

## 핵심 철학
"robust = **잘될 때 잘함이 아니라 좆될 때 덜 죽음**" (GPT 연구원). 상폐(0원)만 아니면 안 죽음 +
성실이 대비 상대평가. 난천은 '정복'(절대수익)이 아니라 '지옥에서도 성실이보다 덜 깨지나'를 본다.

## 설계 v1 (초안 — GPT 1차 검토 반영)
- 매 시험 = 랜덤 시드 → QLD 합성 평행세계 1권 (블록 부트스트랩, `make_world(seed)` 재활용).
- **합성 QLD = QQQ 일별 단순수익률 ×2 복리 − 운용비용** (GPT 정정: 로그×2 아님 — 레버리지 ETF는
  일별 *단순*수익률 2배 목표). 실 QLD는 2006~라 닷컴 없음 → 합성으로 닷컴 커버. **"가상 보스전"이지
  실거래 QLD 복원이 아님**. 실 QLD(2006~)로 근사 검증.
- 참가자: 챔피언 + 성실이(QLD DCA) + 어플삭제단(QLD B&H) + 돼지저금통(현금).
- **평가 = 성실이 대비 score 분포(메인) / 순위는 국면 진단 보조**. 현금·풀매수는 '참가자'보다
  '이 시드가 하락장이냐 상승장이냐 읽는 라벨'로 분리.
  - 핵심 지표(GPT 반영, **하위 꼬리 중심**): 하위 5% score_vs_dca · 꼴찌율 · 최악 MDD ·
    파산(-80%) 도달률 · DCA 승률 · median rank. (샤프는 팻테일 자산서 꼬리 못 잡아 비신뢰)
- **레버리지 난이도 사다리**: 본편 QQQ(1x) → 난천 QLD(2x) → 배틀프론티어 TQQQ(3x, 합성).
- **multiple testing 방지**: 시드를 개발/검증/**봉인** 3분할, 최종 봉인 시드셋은 전략 확정 전 미열람.
- **프레이밍 정직화**: "다목적=robust 증명"(과함) → **"단일목적보다 덜 무너지나 검정"**.

## 검토 반영 누적 (v1.1, GPT+클로드 2중 교차)
- 합성 QLD = **단순수익률 ×2 일별 복리(B방식)** + **시변 차입비용(SOFR)** — 로그×2(A)는 decay 빠짐(금지).
- 평가 = **꼬리 통계량**(CVaR·하위5%·stochastic dominance·국면 층화) — 스칼라 score_vs_dca 자기모순 폐기.
- 기준선 = **QQQ(1x) DCA 바**(QLD DCA는 decay로 본인이 박살=낮은 바) + **정적 50:50 믹스** + **음성 대조군**
  (랜덤 가중치·과적합 전략 — 시험 변별력 자가검증).
- **사전등록 + 홀드아웃 시드 1회** + 파레토 챔피언 선택 규칙 고정 (confirmation bias·다중검정 방지).
- 프레이밍: **난천 단독 robust '증명' 불가 → 정직한 보조 스트레스 시험장**(블록 부트스트랩 정보량 천장 +
  장기메모리 신호 무효측정). 실역사 walk-forward 병행 필수.
- **적대적 시드(제미나이)**: 동적 decay 지옥(+1%/−1% 횡보 장기화) · 상관붕괴(전자산 동시 박살). 지표에
  **Capture Ratio**(상방/하방 캡처) 추가. ⚠️ "블록 63일로 늘리기/regime-switching"은 우리가 이미 막다른
  길 확인(2026-06-19 lab) — 제미나이 §6 처방은 채택 불가.

## 알려진 한계 (2026-06-19 lab 실험서 확인)
- 블록 부트스트랩은 현실 순서·장기추세·국면 비율을 못 살린다(`textbook_blocklen_regime` ·
  `textbook_regime_sampling` 음성). → GPT의 "국면 비율 고정 시험장" 제안은 **블록 방식으론 불가**,
  regime-switching 등 생성모델이라야 가능.

## 설계 v2 — 4중 교차(GPT·클로드·제미나이·코덱스) 합의

**난천 = robust '증명'장 ❌ → 정직한 보조 스트레스 시험장 ✅** ("2배 나스닥 경로에서 단일목적보다
덜 무너지나 검정"). 4명 만장일치 + 우리 lab 발견으로 확정:

- **합성 QLD = 산술(단순)수익률 ×2 일별 복리** (`qld_t = qld_{t-1}·(1 + 2·r_simple − 운용보수 − 시변
  차입비용 − 추적오차)`). 로그×2(A)는 R² 오차로 decay 약화 → **금지**. 닷컴은 QQQ 원본(1999~)에 있어
  QQQ×2면 자동 포함. **실 QLD(2006~)로 오차 검증 — 평균보다 폭락·급반등 구간 편향 방향이 중요.**
- **평가 = 하위 꼬리·paired 중심** (순위는 국면 진단 보조): CVaR 5% · 하위 5% quantile · P(성실이 이김) ·
  최악 MDD · probability of ruin · time under water · **regret**(사후 최선 대비) · **각 기준선과 paired 승률** ·
  Capture Ratio. 스칼라 `score_vs_dca`는 다목적 자기모순이라 메인서 제외.
- **기준선 재설계**: **QQQ(1x) DCA를 진짜 바**(QLD DCA는 decay로 자멸=낮은 바) + **정적 50:50 믹스**
  (능동운용이 보간 이상인가) + **음성 대조군**(랜덤 가중치·과적합 전략 = 시험 변별력 자가검증).
- **다중검정 방지**: 개발/검증/**봉인** 시드셋 3분할(봉인은 1회 개봉·재튜닝 금지) + paired comparison +
  파레토 챔피언 선택 규칙·판정지표·시드수·모델후보 **사전 고정(pre-registration)**.
- **QLD 단일 금지**: 레버리지 그리드(0.5/1/1.5/2/3x 우아하게 열화하나) + 자산 전이(SPY/^NDX) + 상관붕괴
  (2022식) + **신호 지연/누락**(외부 시그널 NaN·지연) + 비용 민감도(0.05~0.5%) + **동적 decay 지옥
  시드**(+1%/−1% 횡보).
- **근본 한계 인정**: 블록 부트스트랩은 정보원천이 QQQ 한 줄(시그널도 같은 데서 생성=비독립) + 장기
  메모리(MA200) 무효측정 + 거시트렌드 파괴. 블록 늘리기·국면 층화·regime-switching은 우리 lab서 막다른
  길 확인. → 난천은 보조, **실역사 walk-forward 병행 필수.**
