# 난천 (Nancheon) 연구 — QLD 2배 지옥 최종보스 시험장

> 챔피언로드 구조: 사천왕(hold-out) → **난천(챔피언전)** → 배틀 프론티어(엔드게임).
> 난천 = 시험마다 랜덤 시드로 생성하는 "QLD(나스닥100 2배 레버리지) 평행세계"에서
> 챔피언이 성실이(DCA)·기준선들 대비 **어떤 국면이 와도 덜 무너지나(robust)** 보는 무대.
> ⚠️ **설계 미확정** — 다중 AI 연구원 교차검토 중 (reviews/).

## 핵심 철학
"robust = **잘될 때 잘함이 아니라 좆될 때 덜 죽음**" (GPT 연구원). 상폐(0원)만 아니면 안 죽음 +
성실이 대비 상대평가. 난천은 '정복'(절대수익)이 아니라 '지옥에서도 성실이보다 덜 깨지나'를 본다.

## 설계 v1 (초안 — GPT 1차 검토 반영)
- 매 시험 = 랜덤 시드 → QLD 합성 평행세계 1권 (블록 부트스트랩, `make_world(seed)` 재활용).
- **합성 QLD = QQQ 일별 단순수익률 ×2 복리 − 운용비용** (GPT 정정: 로그×2 아님 — 레버리지 ETF는
  일별 *단순*수익률 2배 목표). 실 QLD는 2006~라 닷컴 없음 → 합성으로 닷컴 커버. **"가상 보스전"이지
  실거래 QLD 복원이 아님**. 실 QLD(2006~)로 근사 검증.
- 참가자: 챔피언 + 성실이(QLD DCA) + 어플삭제단(QLD B&H) + 돼지저금통(현금).
- **평가 = 성실이 대비 score 분포(메인) / 순위는 국면 진단 보조**. 현금·풀매수는 '참가자'보다
  '이 시드가 하락장이냐 상승장이냐 읽는 라벨'로 분리.
  - 핵심 지표(GPT 반영, **하위 꼬리 중심**): 하위 5% score_vs_dca · 꼴찌율 · 최악 MDD ·
    파산(-80%) 도달률 · DCA 승률 · median rank. (샤프는 팻테일 자산서 꼬리 못 잡아 비신뢰)
- **레버리지 난이도 사다리**: 본편 QQQ(1x) → 난천 QLD(2x) → 배틀프론티어 TQQQ(3x, 합성).
- **multiple testing 방지**: 시드를 개발/검증/**봉인** 3분할, 최종 봉인 시드셋은 전략 확정 전 미열람.
- **프레이밍 정직화**: "다목적=robust 증명"(과함) → **"단일목적보다 덜 무너지나 검정"**.

## 검토 반영 누적 (v1.1, GPT+클로드 2중 교차)
- 합성 QLD = **단순수익률 ×2 일별 복리(B방식)** + **시변 차입비용(SOFR)** — 로그×2(A)는 decay 빠짐(금지).
- 평가 = **꼬리 통계량**(CVaR·하위5%·stochastic dominance·국면 층화) — 스칼라 score_vs_dca 자기모순 폐기.
- 기준선 = **QQQ(1x) DCA 바**(QLD DCA는 decay로 본인이 박살=낮은 바) + **정적 50:50 믹스** + **음성 대조군**
  (랜덤 가중치·과적합 전략 — 시험 변별력 자가검증).
- **사전등록 + 홀드아웃 시드 1회** + 파레토 챔피언 선택 규칙 고정 (confirmation bias·다중검정 방지).
- 프레이밍: **난천 단독 robust '증명' 불가 → 정직한 보조 스트레스 시험장**(블록 부트스트랩 정보량 천장 +
  장기메모리 신호 무효측정). 실역사 walk-forward 병행 필수.
